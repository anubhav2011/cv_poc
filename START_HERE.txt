1. Install dependencies:

   cp .env.example .env
   # Edit .env and add your OPENAI_API_KEY

3. Start the application:
   python -m uvicorn main:app --reload

4. Test the endpoints:

     -F "worker_id=test123" \

   curl http://localhost:8000/form/worker/test123/data
